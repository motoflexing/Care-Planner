# Care Planner Sample Output Snapshots

These examples show the tone and structure produced by the demo services. Every output is educational clinical-support drafting only and requires review by a licensed healthcare professional.

## 1. Condition Name Mode: Diabetes Mellitus

- Possible condition: Diabetes Mellitus
- Confidence: Medium
- Overview: chronic hyperglycemia pattern with complication-prevention focus
- Medical management: assess glycemic status, hydration, complication risk, and clinician-guided therapy needs
- Nursing diagnoses: risk for unstable blood glucose, knowledge deficit, risk for impaired skin integrity
- Red flags: confusion, vomiting, deep breathing, severe dehydration, altered consciousness, severe hypoglycemia symptoms

## 2. Symptom Description Mode: Pneumonia-Like Case

Input pattern:

- Symptoms: fever, cough, shortness of breath
- Duration: 4 days
- Vitals: Temp 38.7 C, SpO2 91%

Example output summary:

- Possible condition: Possible pneumonia or lower respiratory infection
- Confidence: Medium
- Reasoning summary: respiratory symptoms plus infectious features and possible low oxygenation increase concern for lower respiratory involvement
- Differential considerations: bronchitis, viral infection, heart failure exacerbation, pulmonary embolic disease
- Recommended evaluation: pulse oximetry, respiratory exam, chest imaging, labs as indicated
- Emergency red flags: worsening dyspnea, cyanosis, confusion, inability to maintain hydration

## 3. Image Analysis Mode: Dermatitis-Like Rash

Input pattern:

- Image: forearm rash photo
- Context: itchy red rash after new lotion

Example output summary:

- Possible condition: Possible dermatitis or eczema-like rash
- Confidence: Medium
- Key visual findings: patchy erythematous rash-like change, superficial irritation pattern, limited certainty from a single image
- Differential considerations: allergic dermatitis, irritant dermatitis, eczema flare, fungal rash, psoriasis
- Suggested next steps: review exposures, monitor spread, arrange clinician review if painful, febrile, blistering, or worsening
- Emergency red flags: facial swelling, breathing difficulty, widespread blistering, rapidly spreading rash, fever with skin pain or drainage
